Agregar un Tripulante

Para agregar un tripulante en el sistema, sigue los siguientes pasos:

Nota: Asegúrate de que todos los campos estén correctamente llenados antes de intentar agregar un tripulante. Si algún campo está vacío o contiene datos inválidos, el sistema mostrará un mensaje de error y no se agregará el tripulante.

1. Verificar Campos Vacíos

Antes de agregar un tripulante, el sistema verifica que todos los campos requeridos estén llenos. Si algún campo está vacío, se mostrará un mensaje de error.

2. Obtener los Datos del Tripulante

El sistema obtiene los datos del tripulante desde la interfaz de usuario:

  • Nombre del Tripulante: Se obtiene del campo de texto
  • Rango del Tripulante: Se obtiene del comboBox, puede ser -> [Capitán, Ingeniero, Piloto]
  • Nave Asignada: Se obtiene del comboBox de naves disponibles. El sistema extrae el ID de la nave seleccionada.

3. Insertar el Tripulante en el Modelo

Una vez que los datos han sido validados, el tripulante se inserta en la base de datos junto con su rango y el ID de la nave asignada.

4. Limpiar Campos

Después de agregar el tripulante, los campos de la interfaz se limpian para permitir la entrada de nuevos datos.
