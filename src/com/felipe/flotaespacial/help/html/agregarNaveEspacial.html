Agregar una Nave Espacial

Para agregar una nave espacial en el sistema, sigue los siguientes pasos:

Nota: Asegúrate de que todos los campos estén correctamente llenados antes de intentar agregar una nave. Si algún campo está vacío o contiene datos inválidos, el sistema mostrará un mensaje de error y no se agregará la nave.

1. Verificar Campos Vacíos

Antes de agregar una nave, el sistema verifica que todos los campos requeridos estén llenos. Si algún campo está vacío, se mostrará un mensaje de error.

2. Obtener los Datos de la Nave

El sistema obtiene los datos de la nave desde la interfaz de usuario:

  • Nombre de la Nave: Se obtiene del campo de texto
  • Clase de la Nave: Se obtiene del comboBox, puede ser -> [Exploración, Combate, Transporte]
  • Capacidad de la Nave: Se obtiene del campo de texto, tiene que ser numérico

3. Insertar la Nave en el Modelo

Una vez que los datos han sido validados, la nave se inserta en la base de datos

4. Limpiar Campos y Actualizar la Interfaz

Después de agregar la nave, los campos de la interfaz se limpian y el comboBox de naves se actualiza para reflejar los cambios.
