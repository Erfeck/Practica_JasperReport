Agregar una Misión

Para agregar una misión en el sistema, sigue los siguientes pasos:

Nota: Asegúrate de que todos los campos estén correctamente llenados antes de intentar agregar una misión. Si algún campo está vacío o contiene datos inválidos, el sistema mostrará un mensaje de error y no se agregará la misión.

1. Verificar Campos Vacíos

Antes de agregar una misión, el sistema verifica que todos los campos requeridos estén llenos. Si algún campo está vacío, se mostrará un mensaje de error.

2. Obtener los Datos de la Misión

El sistema obtiene los datos de la misión desde la interfaz de usuario:

  • Descripción de la Misión: Se obtiene del área de texto.
  • Estado de la Misión: Se obtiene del comboBox, puede ser -> [Completada, En Progreso, Fallida]
  • Nave Asignada: Se obtiene del comboBox de naves disponibles. El sistema extrae el ID de la nave seleccionada.

3. Insertar la Misión en el Modelo

Una vez que los datos han sido validados, la misión se inserta en la base de datos junto con su estado y el ID de la nave asignada.

4. Limpiar Campos

Después de agregar la misión, los campos de la interfaz se limpian para permitir la entrada de nuevos datos.
